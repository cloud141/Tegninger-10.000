Spilleregler for 10.000

Oprindelsen til terningspillet 10.000 er ukendt. Det er dog helt sikkert at dette spil som så mange andre har forskellige regler afhængig af hvor i landet man kommer fra.

På Terningspil.dk bruger vi følgende regler: Man spiller om at nå 10.000 point først.

Sådan får du point i spillet 10000:

  • En ener giver 100 point
  • En femmer giver 50 point
  • Tre ens giver antal øjne på terningen gange 100. F.eks. giver 3 4’ere 400 point.
  • Fire ens giver som 3 ens X 2. F.eks. giver 4 3’ere 300 X 2.
  • Fem og seks ens giver som 3 ens X 2 for hver ekstra terning. F.eks. giver 5 2’ere 200 X 2 X 2. 6 2’ere giver 200 X 2 X 2 X 2.
  • Enere er anderledes: tre enere giver 1000, fire enere giver 2000, fem enere giver 4000 og seks enere giver 10.000 point.

Hvordan tælles point sammen?

Hver gang man har slået, skal man mindst lægge én pointgivende terning til side.
